# 📔 Manual de Usuario - Plataforma de Servicios

Este manual describe el funcionamiento y uso de la **Plataforma digital para la oferta y demanda de servicios profesionales en Colonias Unidas**.

---

## 1. Interfaz Principal y Búsqueda
Al ingresar a la plataforma, se presentará la pantalla de inicio. Los usuarios pueden utilizar la barra de búsqueda central para localizar profesionales por su área de especialidad.

![Vista Principal](capturas/vista_principal.png)

---

## 2. Control de Acceso (Autenticación)

### 2.1 Inicio de Sesión
Para interactuar con la plataforma, el usuario registrado debe introducir sus credenciales (Correo Electrónico y Contraseña).

![Inicio de Sesión](capturas/login.png)

### 2.2 Registro de Nuevos Usuarios
Si no dispone de una cuenta, puede registrarse completando el formulario con sus datos personales, asignando una contraseña y seleccionando su rol en el sistema.

![Registro de Usuario](capturas/registro.png)

---

## 3. Módulos y Perfiles de Usuario

### 3.1 Vista del Profesional
Una vez autenticado, el profesional cuenta con un perfil público donde se muestran sus datos de contacto, especialidad y los servicios ofertados para la comunidad de Colonias Unidas.

![Perfil Profesional](capturas/perfil.png)
